# Upload Product File

Use this API to upload non-image files such as:

- Videos to enhance product listings
- PDFs for certifications, reports, or compliance documents

This helps meet TikTok Shop requirements for listing restricted products and improves the shopping experience.

> **Note:**
>
> - All non-image files must be uploaded through this API.
> - You must **store the response body** to retrieve the file ID or URL, which is required when associating the file with a product during creation or editing.

## Requirements

- A valid seller **Access Token**
- Use `product.createCustomBrands()` from the SDK

**Official documentation:**  
[TikTok Shop – Upload Product File](https://partner.tiktokshop.com/docv2/page/upload-product-file-202309)

## Body Parameters

| Property | Type   | Description                                                                                                                                                                                                                                                                                                                                                            |
| -------- | ------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| data     | file   | The local file to be uploaded. <br>**Required** <br><br>**Notes:** <br>• Formats: PDF, MP4, MOV, MKV, WMV, WEBM, AVI, 3GP, FLV, MPEG <br>• Max PDF size: 20MB <br>• Max video size: 100MB <br>• Video aspect ratio: 9:16 to 16:9 <br><br>**Recommended for product videos:** <br>• Aspect ratio: 1:1 <br>• Resolution: HD 720p or higher <br>• Duration: 20–60 seconds |
| name     | string | The name of the file, including the extension (e.g., `certification.pdf`). <br>**Required** <br><br>**Notes:** <br>• Use only one period before the extension <br>• Do not start with symbols <br>• Avoid spaces                                                                                                                                                       |

## Usage Example

```ts
import { TikTokShopSDK, TikTokAPIError } from 'tiktok-shop-sdk';

import path from 'path';
import fs from 'fs';

const sdk = new TikTokShopSDK({
  appKey: process.env.TIKTOK_APP_KEY!,
  appSecret: process.env.TIKTOK_APP_SECRET!,
});

try {
  // Set Access Token
  sdk.setAccessToken(process.env.TIKTOK_APP_ACCESS_KEY!);

  const filePath = path.join(process.cwd(), 'images/fnatic.png');
  const fileBuffer = fs.readFileSync(filePath);

  const response = await sdk.product.uploadProductFile({
    data: fileBuffer,
    name: 'fnatice.png',
  });

  console.log(response);
} catch (error) {
  if (error instanceof TikTokAPIError) {
    console.error('TikTok API Error:', error.message);
    console.error('Status Code:', error.code);
    console.log('Request Id: ', error.request_id);
  } else {
    console.error('Unexpected error:', error);
  }
}
```

## Response Descriptions

| Field      | Type   | Description                                                                                          |
| ---------- | ------ | ---------------------------------------------------------------------------------------------------- |
| code       | int    | Status code indicating success or failure of the API response.                                       |
| message    | string | Message describing the result. Includes reasons for failure if applicable.                           |
| request_id | string | Unique identifier for tracking the request.                                                          |
| data       | object | Contains specific return information.                                                                |
| └─ id      | string | Unique file ID generated by TikTok Shop. Use this when linking the file to a product.                |
| └─ url     | string | Direct URL to access and view the uploaded file.                                                     |
| └─ name    | string | The name of the uploaded file.                                                                       |
| └─ format  | string | File format. Possible values: `PDF`, `MP4`, `MOV`, `MKV`, `WMV`, `WEBM`, `AVI`, `3GP`, `FLV`, `MPEG` |

## Error Handling

All API errors throw **TikTokAPIError**, which provides:

- **message**
- **code**
- **request_id**

Full reference:  
[TikTok Common Errors](https://partner.tiktokshop.com/docv2/page/upload-product-file-202309#Error_Code)
